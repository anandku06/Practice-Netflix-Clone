## Run Locally

Clone the project

```bash
  git clone https://anandku06.github.io/Practice-Netflix-Clone/
```

Go to the project directory

```bash
  cd Practice-Netflix-Clone
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
